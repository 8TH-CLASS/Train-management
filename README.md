#                                                                     TRAIN MANAGEMENT SYSTEM

A Java-based backend project simulating a Train Management System with core functionalities like train listing, searching, and basic booking flow using Object-Oriented Programming principles.



Features

- Train data management (stations, routes, schedules)
- Search trains between stations
- Basic booking flow simulation
- Object-Oriented design (Classes, Encapsulation, etc.)
- Console-based Java application



 Tech Stack

- Java
- OOP Concepts
- IntelliJ IDEA (recommended)



Project Structure

- Main.java → Entry point of the application  
- Train / Station / Booking classes → Core models  
- Supporting logic for search and booking flow
